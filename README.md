# Portfolio Website

[https://aablty.github.io](https://aablty.github.io)

## Stack

- React 18 (TS)
- TailwindCSS 
- Supabase

## Installation

```bash
git clone https://github.com/aablty/aablty.github.io.git
cd aablty.github.io
npm install
cp .env.example .env
```

## Commands

```bash
npm run dev      # Dev Preview
npm run build    # Build
npm run deploy   # Deploy to GitHub Pages
```
